git มีประโยชนน์อย่างไร
- ช่วยให้ง่ายในการจัดการ source code
- ง่ายในการแบ่งปัน source code
- มองเห็นการเปลี่ยนแปลง code ได้ง่าย
- เมื่อ code ใหม่มีปัญหา สามารถ ถอยกลับไปยัง version ก่อนหน้าได้
- ทำให้การจัดแก้ code กับเพื่อน ได้ง่าย
- ทำให้ไม่เสียเวลาในการจัดการหา Error
- เมื่อเครื่องมีปัญหา ก็มี code อยู่ที่ server
- ไม่ต้องพก หรือ copy ผ่าน usb
